# GenAI Learning Journal

(This repository documents my learning journey in Generative AI and NLP.
Some reflections and articles related to these experiments are shared in Spanish on LinkedIn.)

This repository documents my learning journey in Generative AI and Natural Language Processing (NLP).

After more than 20 years working in technology — first as a developer and later in leadership and Agile roles — I decided to deepen my understanding of AI and its practical applications in business.

This space serves as a personal journal where I explore concepts, run experiments, and reflect on how AI techniques can generate real-world insights.

## What you will find here

This repository includes:

- Experiments with NLP tools and techniques
- Notes on concepts related to Generative AI
- Small projects using libraries such as:
  - NLTK
  - VADER
  - Hugging Face Transformers
- Reflections on how these technologies can be applied to business problems

## Topics explored

Some of the topics covered include:

- Sentiment analysis
- Topic detection
- Text processing with NLP
- Transformers and modern language models
- Practical applications of AI for customer insights

## Why this repository exists

AI is evolving rapidly, and I believe the most valuable perspective comes from combining:

- Technical understanding
- Product thinking
- Business context

This journal is my way of learning in public and documenting that intersection.
